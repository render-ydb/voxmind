# VoxMind 使用指南

> 本地离线语音输入工具，支持中文、英文、日语、韩语、粤语自动识别

## 系统要求


| 项目   | 要求                                      |
| ---- | --------------------------------------- |
| 操作系统 | macOS 11.0 (Big Sur) 或更高                |
| 芯片架构 | Apple Silicon (M1/M2/M3/M4) 和 Intel 均支持 |
| 磁盘空间 | 约 1.5 GB（应用 + 模型）                       |
| 网络   | 仅首次启动下载模型时需要，之后完全离线运行                   |


## 安装

1. 下载 `voxmind-1.0.0-universal.dmg`
2. 双击打开 DMG，将 VoxMind 拖入 Applications 文件夹
3. 首次打开需要右键点击 VoxMind.app → **打开** → 确认打开（因未签名会被 Gatekeeper 拦截）

> 或在终端执行 `xattr -cr /Applications/VoxMind.app` 后双击打开

## 首次启动

启动后会自动完成以下流程：

1. **请求权限** — 弹窗请求麦克风和辅助功能权限（必须允许，否则无法使用）
2. **下载语音模型** — 自动下载 ASR 模型（约 229 MB），进度条会显示在下载窗口
3. **就绪** — 下载完成后，屏幕右下角出现悬浮球，菜单栏出现柴犬图标

### 权限设置


| 权限   | 用途              | 授权位置                 |
| ---- | --------------- | -------------------- |
| 麦克风  | 录音进行语音识别        | 系统设置 → 隐私与安全性 → 麦克风  |
| 辅助功能 | 模拟按键粘贴文本、检测当前应用 | 系统设置 → 隐私与安全性 → 辅助功能 |


## 基本使用

### 语音输入

1. 将光标放在任意输入框中（如微信、飞书、VSCode、浏览器等）
2. **点击悬浮球** 或按 **⌘ + Shift + Space** 开始录音
3. **再次点击悬浮球** 或再按 **⌘ + Shift + Space** 停止录音
4. 识别完成的文字会自动粘贴到之前聚焦的应用中

> 整个过程完全离线，语音数据不会上传到任何服务器

### 悬浮球状态


| 状态  | 外观             | 说明            |
| --- | -------------- | ------------- |
| 待命  | 灰色半透明球 + 麦克风图标 | 等待点击或快捷键      |
| 录音中 | 红色球 + 脉冲动画     | 正在录音，实时显示识别文字 |
| 处理中 | 灰色球 + 旋转加载图标   | 正在转写/校正文本     |
| 下载中 | 半透明灰色球（不可点击）   | 正在下载模型        |


### 移动悬浮球

按住悬浮球拖动到屏幕任意位置，松手后位置会自动保存，下次启动恢复。

## 菜单栏选项

点击菜单栏的柴犬图标，可以看到以下选项：

### LLM Correction（LLM 文本校正）

- 默认关闭
- 首次开启时会下载 Qwen 2.5 1.5B 模型（约 1 GB）
- 开启后，语音识别的结果会经过本地 LLM 校正，修复同音错字、标点符号、中英混合等问题
- 示例：`我要用 python 写一个 jason 解析器` → `我要用 Python 写一个 JSON 解析器`

### Open at Login（开机自启）

- 开启后 macOS 启动时自动运行 VoxMind

### Auto Send（自动发送）

- 开启后，粘贴文字后会自动按下回车键
- 适合在聊天软件（微信、飞书、Slack 等）中直接发送消息
- 粘贴后会等待 1 秒再按回车，确保文字已输入完毕

### Open Data Folder（打开数据目录）

- 在 Finder 中打开数据目录，可查看下载的模型和历史记录

### Quit（退出）

- 完全退出 VoxMind

## 快捷键


| 快捷键               | 功能                      |
| ----------------- | ----------------------- |
| ⌘ + Shift + Space | 开始/停止录音（全局有效，任何应用下都可使用） |


## 支持的语言

语音识别会自动检测语言，无需手动切换：

- 中文（普通话）
- 英文
- 日语
- 韩语
- 粤语

支持中英混合语音输入。

## 数据存储

所有数据存储在本地，位置如下：

```
~/Library/Application Support/voxmind/
├── settings.json              # 用户设置
└── data/
    ├── models/                # AI 模型文件
    │   ├── silero_vad.onnx                         # VAD 模型 (~632 KB)
    │   ├── sherpa-onnx-sense-voice-.../             # 语音识别模型 (~228 MB)
    │   └── llm/qwen2.5-1.5b-instruct-q4_k_m.gguf  # LLM 模型 (~1 GB, 可选)
    └── history/               # 转写历史记录
        └── YYYY-MM-DD/
            └── transcripts.json
```

- 历史记录上限 500 MB，超出后自动清理最早的记录
- **更新应用不会删除这些数据**，放心覆盖安装新版本

## 更新版本

1. 下载新版本的 DMG
2. 退出正在运行的 VoxMind（菜单栏 → Quit）
3. 将新版本拖入 Applications 覆盖旧版本
4. 重新打开即可，所有模型和历史记录保留

## 常见问题

### 悬浮球不见了？

VoxMind 没有 Dock 图标，请检查菜单栏是否有柴犬图标。如果没有，可能应用未启动，请从 Launchpad 或 Applications 文件夹重新打开。

### 点击悬浮球没反应？

检查辅助功能权限是否已授予：系统设置 → 隐私与安全性 → 辅助功能 → 确认 VoxMind 已勾选。修改权限后需要重启应用。

### 录音后没有文字输出？

检查麦克风权限是否已授予：系统设置 → 隐私与安全性 → 麦克风 → 确认 VoxMind 已勾选。

### 文字粘贴到了错误的应用？

VoxMind 会在开始录音时记住当前聚焦的应用。请确保在点击悬浮球前，光标已在目标输入框中。使用快捷键 ⌘ + Shift + Space 可以更精确地控制。

### 需要联网吗？

仅首次启动下载模型时需要网络。下载完成后，所有功能完全离线运行，不会向任何服务器发送数据，所有数据保存在本地，保证数据安全。
